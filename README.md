# 📱 Quote of the Day App

## 📌 Overview

Quote of the Day App is an Android application developed as part of the **CodSoft Android App Development Internship**.

The application displays motivational quotes, allows users to save their favorite quotes, and provides an easy way to share them with friends and family.

---

## ✨ Features

- 📖 Display inspirational quotes
- 🔄 Refresh to get a new quote
- ❤️ Save favorite quotes
- 📂 View saved quotes
- 📤 Share quotes via social media and messaging apps
- 📱 Clean and user-friendly UI

---

## 🛠️ Tech Stack

- Android Studio
- Java/Kotlin
- XML
- SharedPreferences / Room Database
- Material Design Components
